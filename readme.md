# Webpack Template

## Features

- Basic files support (HTML, CSS, Stylus, JS, Fonts, Images)
- Extracted CSS
- Transpilled JS (with Babel)
- Sourcemaps
- Clean dist
- Dev server (with local network support)
- Static Files
- Cache breaker

## Instructions

Installe dependencieis

```npm i```

Launch dev server

```npm run dev```

Build for production

```npm run build```
